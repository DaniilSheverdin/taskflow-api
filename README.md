# [pet-project] taskflow-api

Асинхронный Task Manager с JWT-аутентификацией и ролями.

Реализована авторизация и аутентификация, описанная в этой
статье: https://gist.github.com/zmts/802dc9c3510d79fd40f9dc38a12bccfc

## Features

- [x] Асинхронная обработка запросов
- [ ] RESTful API для управления задачами (task manager)
- [x] JWT-аутентификация с access и refresh токенами
- [x] Механизм защиты от кражи refresh токена
- [ ] Ролевая модель доступа
- [ ] CRUD-операции для задач с фильтрацией
- [ ] Пагинация и фильтрация списков задач
- [ ] Поддержка уведомлений (WebSockets)
- [ ] Интеграция с фоновыми задачами (например, через BackgroundTasks)
- [x] Логирование ошибок и базовая валидация данных
