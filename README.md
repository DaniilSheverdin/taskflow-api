# [pet-project] taskflow-api

Асинхронный Task Manager с JWT-аутентификацией и ролями

## Features

- [ ] Асинхронная обработка запросов
- [ ] RESTful API для управления задачами (task manager)
- [ ] JWT-аутентификация с access и refresh токенами
- [ ] Ролевая модель доступа (RBAC): User, Manager, Admin
- [ ] CRUD-операции для задач с фильтрацией по статусу, приоритету и назначению
- [ ] Пагинация и фильтрация списков задач
- [ ] Поддержка уведомлений (опционально, via email или WebSockets)
- [ ] Интеграция с фоновыми задачами (например, через BackgroundTasks)
- [ ] Логирование ошибок и базовая валидация данных
